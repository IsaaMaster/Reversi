About

This project is a remake of the popular game Reversi, allowing users from around the world to chat and play against each other online, among many other features. It was made using HTML, CSS (Bootstrap 5), Vanilla JS, Sockets.io, Node.js and deployed to Heroku. The source code can be seen here: Github.

If you checking out the website, you may need to open the game in a second tab and join the lobby from both tabs in order to play against yourself. You could also try accessing the website from another device and playing against yourself.
